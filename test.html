        How this works
        

          The iframe loads simulide.html. Once the WASM module
          is ready, this page reaches iframe.contentWindow.Module
          .SimulIDEBridge.instance() and wires
          onStateChange / onError /
          onCompiled. Buttons call the corresponding C++
          methods through embind. Same-origin only — both files must be
          served from the same host/port.
